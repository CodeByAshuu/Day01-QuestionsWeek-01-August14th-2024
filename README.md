Enter the second string: World

Sample Output:- Concatenated string: HelloWorld

Sample Input:- Enter grade 1: 85
Enter grade 2: 90
Enter grade 3: 78
Enter grade 4: 92
Enter grade 5: 88

Sample Output:- The average grade is: 86.6
